# CIA Factbook
 This is a tool written in python to explore the CIA Factbook and store it in a relational database (SQLite), it provides basic intelligence on the history, people, government, economy, energy, geography, communications, transportation, military, terrorism, and transnational issues for 266 world entities.

### Installation
`pip3 install PyInquirer pyfiglet prettytable ptable`
